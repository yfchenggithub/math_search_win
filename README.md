# math_search_win

Windows 本地离线版《高中数学二级结论搜索系统》MVP

## 当前技术路线

- Qt Widgets：主框架
- Qt WebEngine：详情页正文渲染
- Qt SVG：搜索/收藏卡片公式预览
- C++17 + CMake + MSVC
- VSCode 主开发，Qt Creator/Designer 辅助

## 当前阶段

第一阶段：工程骨架 + 六页静态壳子

## 目录说明

- src/：源码
- resources/：图标、样式、web 模板
- data/：本地内容与索引数据
- license/：授权文件
- cache/：收藏、最近搜索、设置缓存
- docs/：文档
